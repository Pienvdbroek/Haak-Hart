# Haak&Hart
Haak & Hart Grafisch Examen Pien & Luuk

Inloggen

Je hoeft geen bestaand account te gebruiken.
Druk bij het inloggen direct op Aanmelden om verder te gaan.

Installatie

Open de terminal in de projectmap en voer de volgende commands uit:

composer install
npm install

Start daarna de frontend:

npm run dev

Open een tweede terminal en start de Laravel server:

php artisan serve

De website is daarna lokaal beschikbaar via:

http://127.0.0.1:8000
